# 深度强化学习简介 [[introduction-to-deep-reinforcement-learning]]

<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit1/thumbnail.jpg" alt="Unit 1 thumbnail" width="100%">


欢迎来到人工智能中最有趣的话题： **深度强化学习。**

深度 RL 是一种机器学习，代理可以在其中学习 **如何表现** 在一个环境中 **通过执行动作** 和 **看到结果。**

在这第一单元中， **您将学习深度强化学习的基础。**


然后，你会 **训练您的深度强化学习代理（月球着陆器）正确登陆月球** 使用 <a href="https://stable-baselines3.readthedocs.io/en/master/"> Stable-Baselines3 </a>，一个深度强化学习库。


<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit1/lunarLander.gif" alt="LunarLander">

最后，你会 **将此训练有素的代理上传到 Hugging Face Hub 🤗，这是一个免费的开放平台，人们可以在其中共享 ML 模型、数据集和演示。**

这是必不可少的 **掌握这些要素** 在深入实施深度强化学习代理之前。本章的目标是为您提供坚实的基础。


在本单元之后，在奖励单元中，您将 **能够训练 Huggy the Dog 🐶 拿起棍子并和他一起玩 🤗**.
