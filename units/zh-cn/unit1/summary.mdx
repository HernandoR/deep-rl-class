# 总结 [[summary]]

这是很多信息！我们总结一下：

- 强化学习是一种从动作中学习的计算方法。我们构建一个可以从环境中学习的代理 **通过反复试验与它互动** 并获得奖励 (消极或积极) 作为反馈。

- 任何 RL 代理的目标都是最大化其预期累积奖励 (也称为预期回报) 因为 RL 是基于 **奖励假设**，这就是 **所有目标都可以描述为期望累积奖励的最大化。**

- RL 进程是一个循环，输出一系列 **状态、行动、奖励和下一个状态。**

- 计算预期累积奖励 (预期回报)，我们对奖励打折：来得更快的奖励 (在游戏开始时) **发生的可能性更大，因为它们比长期的未来奖励更可预测。**

- 要解决 RL 问题，您需要 **找到最优策略**。该政策是您代理人的“大脑”，它会告诉我们 **给定一个状态要采取什么行动。** 最优的政策是 **为您提供最大化预期回报的行动。**

- 有两种方法可以找到最佳策略：
    1. 通过直接训练您的策略： **基于政策的方法。**
    2. 通过训练一个价值函数，该函数告诉我们代理在每个状态下将获得的预期回报，并使用该函数来定义我们的策略： **基于价值的方法。**

- 最后，我们谈论 Deep RL 因为我们介绍 **深度神经网络来估计要采取的行动 (基于政策的) 或估计一个状态的价值 (基于价值的)** 因此得名“深”。
