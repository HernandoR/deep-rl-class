<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit1/RL_process_game.jpg" alt="The RL process" width="100%">

- 我们的代理收到 **状态  \\(S_0\\)** 从 **环境** — 我们收到游戏的第一帧 (环境).
- 基于此 **状态 \\(S_0\\),** 代理采取 **行动 \\(A_0\\)** — 我们的特工将移至右侧。
- 环境转到一个 **新的** **状态 \\(S_1\\)** — 新框架。
- 环境给了一些 **奖励 \\(R_1\\)** 致特工——我们还没死 *(正面奖励+1)*.

该 RL 循环输出一系列 **状态、行动、奖励和下一个状态。**

<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit1/sars.jpg" alt="State, Action, Reward, Next State" width="100%">

代理的目标是最大化其累积奖励， **称为预期回报。**

## 奖励假设：强化学习的中心思想 [[reward-hypothesis]]

⇒ 为什么代理人的目标是最大化预期回报？

因为RL是基于 **奖励假设**，即所有目标都可以描述为 **期望回报最大化** (预期累积奖励).

这就是为什么在强化学习中， **拥有最好的行为，** 我们的目标是学会采取行动 **最大化预期累积奖励。**


## 马尔可夫性质 [[markov-property]]

在论文中，您会看到 RL 进程被称为 **马尔可夫决策过程** (MDP).

我们将在下面的单元中再次讨论马尔可夫性质。但如果你今天需要记住一些关于它的事情，那就是：马尔可夫性质意味着我们的代理需要 **只有当前状态才能决定** 采取什么行动以及 **不是所有状态和行动的历史** 他们之前拿过。

## 观察/状态空间 [[obs-space]]

观察/状态是 **我们的代理从环境中获取的信息。** 对于视频游戏来说，它可以是一个帧 (截图)。对于交易代理来说，它可以是某种股票的价值等。

之间有一个区别 *观察* 和 *状态*，但是：

- *国家*: 是 **对世界现状的完整描述** (没有隐藏信息)。在充分观察的环境中。

这些动作可以来自 *离散的* 或 *连续空间*:

## 奖励和折扣 [[rewards]]

奖励在 RL 中至关重要，因为它是 **唯一的反馈** 对于代理。多亏了它，我们的代理才知道 **所采取的行动是否良好。**

每个时间步的累积奖励 **t** 可以写成：

<figcaption>累积奖励等于序列中所有奖励的总和。
</figcaption>
</figure>

这相当于：

<figure>
<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit1/rewards_2.jpg" alt="Rewards">
<figcaption>累计奖励=rt+1 (rt+k+1 = rt+0+1 = rt+1)+ rt+2 (rt+k+1 = rt+1+1 = rt+2) + ...
</figcaption>
</figure>

然而，在现实中， **我们不能就这样添加它们。** 奖励来得更早 (在游戏开始时) **更有可能发生** 因为它们比长期的未来奖励更可预测。

假设你的智能体是这只小老鼠，每一步可以移动一个方块，而你的对手是猫 (也可以移动)。老鼠的目标是 **在被猫吃掉之前吃掉最大量的奶酪。**

<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit1/rewards_3.jpg" alt="Rewards" width="100%">

正如我们在图中看到的， **吃我们附近的奶酪的可能性比吃离猫近的奶酪的可能性更大** (我们离猫越近，它就越危险).

因此， **靠近猫的奖励，即使它更大 (更多奶酪)，会更优惠** 因为我们不确定我们能不能吃它。

为了折扣奖励，我们按如下方式进行：

1. 我们定义一个称为伽玛的贴现率。 **它必须介于 0 和 1 之间。** 大部分时间之间 **0.95 和 0.99**.
- gamma 越大，折扣越小。这意味着我们的代理 **更关心长期回报。**
- 另一方面，gamma越小，折扣就越大。这意味着我们的 **代理人更关心短期回报 (最近的奶酪).**

2. 然后，每个奖励将按 gamma 折扣到时间步长的指数。随着时间步长的增加，猫离我们越来越近， **所以未来的回报发生的可能性越来越小。**
