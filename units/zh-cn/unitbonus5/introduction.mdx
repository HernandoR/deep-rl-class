## 目标：

- 通过使用人工录制的专家演示来训练代理完成迷你游戏环境，了解如何使用 Godot RL Agents 进行模仿学习。

## 先决条件和要求：

- 建议您完成上一章 ([Godot RL Agents](https://huggingface.co/learn/deep-rl-course/unitbonus3/godotrl)) 在开始本教程之前，
- 建议熟悉 Godot，尽管完成本教程不需要任何 gdscript 编码知识，
- Godot 具有 .NET 支持 (经测试可与 [4.3.dev5.NET](https://godotengine.org/article/dev-snapshot-godot-4-3-dev-5/)，也可能适用于较新的版本),
- Godot RL Agents (你可以使用 `pip install godot-rl` 在 venv/conda 环境中),
- [仿库](https://huggingface.co/learn/deep-rl-course/unitbonus5/train-our-robot),
- 时间：约 1-2 小时完成项目和培训。它可能超出此范围，具体取决于所使用的硬件。
