# 奖励：学习使用 Unity 和 MLAgents 创建自己的环境

**您可以使用 Unity 和 MLAgents 创建自己的强化学习环境**。使用 Unity 等游戏引擎一开始可能会令人生畏，但您可以采取以下步骤来顺利学习。

## 步骤1：了解如何使用Unity

- 学习 Unity 的最佳方法是 [“用代码创建”课程](https://learn.unity.com/course/create-with-code)：这是一系列适合初学者的视频 **您将使用 Unity 创建 5 个小游戏**.

## 第 2 步：使用本教程创建最简单的环境

- 然后，当您知道如何使用 Unity 时，您可以创建您的 [使用本教程的第一个基本 RL 环境](https://github.com/Unity-Technologies/ml-agents/blob/release_20_docs/docs/Learning-Environment-Create-New.md).

## 第 3 步：迭代并创建良好的环境

- 现在您已经创建了第一个简单的环境，您可以使用以下命令迭代到更复杂的环境 [MLAgents 文档 (特别是设计代理和代理部分)](https://github.com/Unity-Technologies/ml-agents/blob/release_20_docs/docs/)
- 此外，您还可以参加此免费课程 [“创造一个蜂鸟环境”](https://learn.unity.com/course/ml-agents-hummingbirds) 通过 [亚当·凯利](https://twitter.com/aktwelve)


玩得开心！如果您创建了自定义环境，请毫不犹豫地将它们分享给 `#rl-i-made-this` Discord 频道。
