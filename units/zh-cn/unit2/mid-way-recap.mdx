# 中途回顾 [[mid-way-recap]]

在深入了解 Q-Learning 之前，我们先总结一下刚刚学到的内容。

我们有两种类型的基于值的函数：

- 状态值函数：输出预期收益，如果 **代理从给定的状态开始，并在此后永远按照策略行事。**
- 动作值函数：输出预期回报，如果 **代理从给定状态开始，在该状态采取给定操作** 然后永远按照该政策行事。
- 在基于价值的方法中，而不是学习政策， **我们手动定义策略** 我们学习一个价值函数。如果我们有一个最优价值函数，我们 **就会有最优的政策。**

更新价值函数的方法有两种：

- 与 *蒙特卡罗方法*，我们从一个完整的情节中更新价值函数，所以我们 **使用本集的实际折扣回报。**
- 与 *TD学习方法，* 我们从一步更新价值函数，替换未知数 \\(G_t\\) 与 **称为 TD 目标的估计回报。**
