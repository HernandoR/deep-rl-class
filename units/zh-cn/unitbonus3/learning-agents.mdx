# 虚幻学习代理简介

[学习代理](https://dev.epicgames.com/community/learning/tutorials/8OWY/unreal-engine-learning-agents-introduction) 是 Unreal Engine (UE) 插件可以让你 **使用机器学习训练 AI 角色 (机器学习) 在虚幻中**.

这是一个令人兴奋的新插件，您可以使用 Unreal Engine 创建独特的环境并培训您的代理。

## 案例1：我对Unreal Engine和Unreal Engine的初学者一无所知
如果您是 Unreal Engine 的新手，请不要害怕！我们列出了您需要学习的两门课程才能使用学习代理：

1. 掌握基础知识：从观看本课程开始 [您在 Unreal Engine 5 中的第一个小时](https://dev.epicgames.com/community/learning/courses/ZpX/your-first-hour-in-unreal-engine-5/E7L/introduction-to-your-first-hour-in-unreal-engine-5)。这门综合课程将 **奠定使用虚幻引擎所需的基础知识**.

2. 深入了解蓝图：探索蓝图的世界，这是 Unreal Engine 的可视化脚本组件。 [本视频课程](https://youtu.be/W0brCeJNMqk?si=zy4t4t1l6FMIzbpz) 将使您熟悉这个重要的工具。

有了基础知识， **您现在已准备好使用学习代理**:

3. 通过以下方式全面了解学习代理 [阅读这份内容丰富的概述](https://dev.epicgames.com/community/learning/tutorials/8OWY/unreal-engine-learning-agents-introduction).

4. [使用学习代理中的强化学习来教汽车驾驶](https://dev.epicgames.com/community/learning/tutorials/qj2O/unreal-engine-learning-to-drive).

5. [使用 Unreal Engine 5.3 学习代理插件检查模仿学习](https://www.youtube.com/watch?v=NwYUNlFvajQ)
