# 值得尝试的有趣环境

在这里，我们提供了一系列有趣的环境，您可以尝试在这些环境中培训您的代理：

## DIAMBRA竞技场

<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit12/diambraarena.png" alt="diambraArena"/>


DIAMBRA Arena 是一个软件包，提供一系列用于强化学习研究和实验的高质量环境。它为流行的街机模拟视频游戏提供了标准接口，并提供了完全符合 OpenAI Gym/Gymnasium 格式的 Python API，使其采用更加顺利和简单。

它支持所有主要操作系统 (Linux、Windows 和 MacOS) 并且可以通过轻松安装 [Python 画中画](https://pypi.org/project/diambra-arena/)。完全免费使用，用户只需在网站上注册即可 [官方网站](https://diambra.ai/register/).

此外，其 [GitHub 存储库](https://github.com/diambra/) 提供了一系列示例，涵盖感兴趣的主要用例，只需几个步骤即可运行。

#### 主要特点

所有环境都是情景强化学习任务，具有离散的动作 (游戏手柄按钮) 以及由屏幕像素加上附加数值数据组成的观察结果 (RAM 值，例如角色生命值条或角色舞台边).

他们都支持单人游戏 (1P) 以及两名玩家 (2P) 模式，使它们成为探索标准 RL、竞争性多智能体、竞争性人类智能体、自我游戏、模仿学习和人机循环的完美资源。

[接口游戏](https://docs.diambra.ai/envs/games/) 被选为最受欢迎的格斗复古游戏之一。虽然共享相同的基本机制，但它们提供了不同的挑战，具有特定的功能，例如不同类型和数量的角色、如何执行连击、生命条充电等。

DIAMBRA Arena 旨在最大限度地提高所有主要强化学习库的兼容性。它本身提供了两个最重要的包的接口： [Stable-Baselines3](https://stable-baselines3.readthedocs.io/en/master/) 和 [Ray RLlib](https://docs.ray.io/en/latest/rllib/index.html)，而Stable Baselines也可用，但已弃用。它们的用法如图所示 [官方文档](https://docs.diambra.ai/) 并在 [DIAMBRA 代理示例存储库](https://github.com/diambra/agents)。它可以以类似的方式轻松地与任何其他包进行交互。

### 竞赛平台

DIAMBRA还提供了与Hugging Face Hub完全集成的竞赛平台，您可以在该平台上提交训练有素的代理，并与全球其他程序员在史诗般的视频游戏锦标赛中一较高下！

它具有一个公共排行榜，其中用户根据其代理在我们不同环境中取得的最佳分数进行排名。

它还提供了根据特工表现解锁酷炫成就的可能性。

提交的特工将接受评估，他们的剧集将在 [DIAMBRA Twitch 频道](https://www.twitch.tv/diambra_ai).

## MineRL

<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit12/minerl.jpg" alt="MineRL"/>


MineRL 是一个 Python 库，提供用于与视频游戏 Minecraft 交互的 Gym 界面，并附有人类游戏玩法的数据集。
这个图书馆每年都会面临挑战。检查 [网站](https://minerl.io/)

要开始使用此环境，请检查以下资源：
- [什么是 MineRL？](https://www.youtube.com/watch?v=z6PTrGifupU)
- [MineRL 的第一步](https://www.youtube.com/watch?v=8yIrWcyWGek)

## 驴车模拟器

<img src="https://huggingface.co/datasets/huggingface-deep-rl-course/course-images/resolve/main/en/unit12/donkeycar.jpg" alt="Donkey Car"/>
Donkey 是一个针对爱好遥控汽车的自动驾驶汽车平台。
该模拟器版本基于Unity游戏平台构建。它使用它们的内部物理和图形，并连接到驴 Python 进程，以使用我们训练的模型来控制模拟驴 (汽车).


要开始使用此环境，请检查以下资源：
- [DonkeyCar 模拟器文档](https://docs.donkeycar.com/guide/deep_learning/simulator/)
- [学会平稳驾驶 (安东尼拉芬的教程)第 1 部分](https://www.youtube.com/watch?v=ngK33h00iBE)
- [学会平稳驾驶 (安东尼拉芬的教程) 第2部分](https://www.youtube.com/watch?v=DUqssFvcSOY)
- [学会平稳驾驶 (安东尼拉芬的教程) 第三部分](https://www.youtube.com/watch?v=v8j2bpcE4Rg)

- 预训练代理：
  - https://huggingface.co/araffin/tqc-donkey-mountain-track-v0
